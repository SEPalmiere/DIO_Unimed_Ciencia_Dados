# processador_imagens_Palmiere

Description. 
The package processador_imagens_Palmiere is used to:
	Processing
	- Histogram matching
	- Structural similarity
	- Resize image

	Utils:
	- Raed image
	- Save image
	- Plot image
	- Plot result
	- Plot histogram

## Installation

```bash
pip install processador_imagens_Palmiere
```
